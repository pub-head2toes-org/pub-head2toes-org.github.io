KISS

Keep It Simple Sam

Simple web by getting back to basics: HTML5 + CSS. Simpe web app server with Node JS using only standard libraries. Simple DB by combining the best of relational and no-SQL DB approaches through simple REST interface. Simple user authentication, and session management.

Head 2 Toes is inviting you to get in the rhythm, keep your cool, and build a momentum.

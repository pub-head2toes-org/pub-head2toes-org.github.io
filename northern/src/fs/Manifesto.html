Storage

Can we have the best of both worlds, SQL and no-SQL?

What is the DB without the API? Can we have the REST API that comes with the storage?

Storage API example:

To get payload addressed with some path:

GET /some/path/

To insert payload addressed with some path:

POST /some/path/ payload body

To update payload addressed with some path:

PUT /some/path/ payload body

Web app

Can we build the web apps just using the HTML and CSS? What can be more basic than that?

Example of web app:

Built with Node JS or Golang standard libraries, using the REST Storage API

Authentication

What can be very simple way of user registration and authentication? Can it be also passwordless?

Example of authentication:

Generate keys in browser, and download them to a local file system.

To authenticate click a button to upload the keys from local file system to produce a signature

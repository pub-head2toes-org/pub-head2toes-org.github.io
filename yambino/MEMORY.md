# Project: Yambino

This is a PWA, that implements presentation of Yahtzee board.

## Impemented Actions

* Draw button
* New button

### Draw button on-click action

When "Draw" button is pressed the random values are selected to up to six dices. "noOfDice" is the max number of dices in use.

Dices are rendered on canvas.

Each dice can be selected by clicking on it. In which case the dice is added to the list of selected dices and rendered in the "canvas-select" canvas field.

Selected dices can be discared and removed from the list by clicking on them.

### New button on-click action

Clicking on the "New" button will reset the selection process.
